# Realty CRM Pro

CRM система для управления недвижимостью и клиентами.

## Запуск локально

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите проект (Frontend + Backend):
   ```bash
   npm start
   ```
   
   Или по отдельности:
   - Frontend: `npm run dev`
   - Backend: `npm run server`

## Развертывание

### Vercel (Frontend)
Проект настроен для развертывания на Vercel.

### VPS (Backend / Full Stack)
Используйте Docker Compose для запуска сервера и базы данных.
